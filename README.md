# LLVM Kaleidoscope Frontend

This project is my implementation of an LLVM frontend for a toy language called 
Kaleidoscope detailed [here](https://llvm.org/docs/tutorial/MyFirstLanguageFrontend/LangImpl01.html#id1).

## Heavily WIP
- Currently implementing the lexer.

## Build Instructions

1. `cmake -B build -S .`
2. `cmake --build build`
3. to run: `./build/Kaleidoscope`
